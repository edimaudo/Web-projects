## Idea Manager
***

### What is this app?

A simple web app that helps keep track of ideas.  

### Core Objective

Learn how to build a crud app

### How it was built

I built it using PHP, MySQL and Bootstrap 3

### Possible extensions
- Ability to add sketches
- Ability to add tags
- Ability to sort the titles or by date
